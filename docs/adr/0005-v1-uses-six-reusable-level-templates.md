# v1.0 Uses Six Reusable Level Templates

UQIQ v1.0 will target six reusable Level Templates across six Level Packs instead of building every Level as a custom one-off: Tap Logic, Drag Logic, Text Trap, Pattern Grid, Memory Flash, and Physics Draw. This keeps 60 Levels feasible while still allowing each Pack to feel distinct through theme, content, timing, layout, scoring, Chaos Modifiers, and a small number of signature custom moments.
