# v1.0 Uses Crash Reporting and Minimal Anonymous Level Events

UQIQ v1.0 will include crash reporting and, at most, minimal anonymous Level Events such as started, completed, failed, DUR'D, and Roast used. The game has no accounts, ads, in-app purchases, or identity tracking in v1.0, so analytics must stay narrow enough to preserve simple privacy disclosures and player trust.
